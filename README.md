# SY Tiimi tekemisen laadun mittaristo – itsearviointilomake

Kevyt verkkopohjainen itsearviointilomake myyjille.

## Muutokset tässä versiossa

- Palautettu aiempi visuaalinen kortti-/valintapohjainen ulkoasu.
- Päivitetty vain mittarien määrittelytekstit käyttäjän antamien versioiden mukaisiksi.
- Fontti asetettu Barlow-fontiksi.
- Vastaanottaja: davis.lipskis@yrittajat.fi

## Käyttö

- Myyjä täyttää nimen ja pisteyttää 5 mittaria.
- Lomake muodostaa valmiin sähköpostipohjan.
- "Avaa sähköpostiluonnos" avaa käyttäjän oman sähköpostiohjelman.
- "Kopioi sähköpostipohja" kopioi saman sisällön leikepöydälle.

## Vercel

Vercel-asetukset:

- Install Command: npm install
- Build Command: npm run build
- Output Directory: dist
- Root Directory: ./
